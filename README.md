## 📌 Project Overview
This project is a **Machine Learning-based web application** that predicts whether a customer is likely to churn or not.

It uses a trained ML model and provides **real-time predictions through an interactive dashboard built with Streamlit**.

---

## 🎯 Key Features

✅ Predict customer churn in real-time  
✅ Clean and interactive dashboard UI  
✅ Feature engineering for better accuracy  
✅ Handles class imbalance using SMOTE  
✅ Displays churn probability and status  
✅ Visual representation using charts  

---

## 🧠 Machine Learning Workflow

1. Data Cleaning & Preprocessing  
2. Feature Engineering (`TotalCharges`)  
3. One-Hot Encoding for categorical variables  
4. Handling imbalance using SMOTE  
5. Model Training (Random Forest)  
6. Model Evaluation  
7. Deployment using Streamlit  

---

## 📊 Input Features

- Tenure  
- Monthly Charges  
- Contract Type  
- Internet Service  
- Payment Method  

---

## 📈 Output

- Churn Probability (0–1)  
- Customer Status (Stay / Churn)  
- Visual Chart Representation
